MERN Stack Interview Assignment – HRMS (Human Resource Management System) 
Assignment Overview 
You are required to develop a simplified but production-oriented HRMS (Human Resource 
Management System) using the MERN Stack (MongoDB, Express.js, React.js, Node.js). 

This assignment is designed to evaluate your: 

• System design and architecture skills  
• Backend API development  
• Database design  
• Authentication & Authorization  
• Business logic implementation  
• React frontend development  
• Code quality and maintainability  
• Problem-solving ability

3 Days (72 Hours) 

Expected effort: 15–25 hours

Late Mark Logic 
Rule 
Office Start Time = 09:30 AM 

If employee punches in after 09:30 AM: 

Late Mark +1 

Monthly Rule 
Every: 



3 Late Marks = 0.5 Day Deduction 

Example: 

Late Marks = 7 
 
3 = Half Day 
3 = Half Day 
 
Remaining = 1 
 
Total Deduction = 1 Day 

 

Leave Adjustment Rule 
Before salary deduction: 

System should check: 

Available Paid Leave? 

If Available 
Deduct from paid leave balance. 

Else 
Salary deduction should apply.
